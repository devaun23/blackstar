# OBSTETRIC COMPLICATIONS

## Preterm Birth Prevention
1. (+) Prior preterm delivery + Contractions → IM 17-hydroxyprogesterone
2. (+) Prior preterm delivery + No contractions (Cervical Insufficiency) → Prophylactic Cerclage
3. (-) Prior preterm delivery + Short cervix <25mm at 16-24wks → Vaginal Progesterone
4. Management by gestational age:
   - <32wks: Betamethasone + Tocolytics (Indomethacin) + Penicillin if GBS+ + Mg Sulfate (neuroprotection)
   - 32-33wks: Betamethasone + Tocolytics + Penicillin
   - 34-36wks: ± Betamethasone + Penicillin; Tocolytics avoided (Indomethacin: Oligohydramnios/PDA closure; Nifedipine: Maternal hypotension)

## PPROM (<37wks prior to labor)
- RF: Prior PPROM, GU infection (asymptomatic bacteriuria), BV, Cervical insufficiency, Polyhydramnios
- Dx: Vaginal pooling, Nitrazine-positive (blue) fluid, Ferning on microscopy
- <34wks Reassuring → Latency Abx + Betamethasone; <34wks Non-reassuring or ≥34wks → Delivery
- Complications: Preterm labor, Intraamniotic infection, Abruption, Cord prolapse

## 3rd Trimester Bleeding

### Abruptio Placentae (Painful)
1. Premature separation; Peaks 24-26wks
2. RF: #1 HTN; Uterine overdistension (twins, polyhydramnios); Cocaine/Tobacco; Trauma; Prior abruption
3. CF: High-frequency contractions, ROCK HARD UTERUS, Painful bleeding; Retroplacental concealed hemorrhage possible
4. Eval: CBC, PT/PTT, Fibrinogen (most sensitive), Rh typing, Rosette test → KB test
5. Tx: C-section; <34wks → CST; Hemorrhagic shock → IVF + Blood products + Left lateral decubitus
6. Complications: DIC (tissue factor from decidual bleeding), Fetal hypoxia/demise

### Placenta Previa (Painless)
1. RF: Prior C-section, Prior previa, Multiple gestation, AMA
2. CF: 3rd TM painless vaginal bleeding (maternal origin initially → reactive fetal tracing)
3. Dx: Transabdominal → Transvaginal US
4. Management: Pelvic rest (no intercourse/digital exam); Speculum OK; Repeat US 3rd TM
5. Persistent Previa → C-section 36-37wks

### Vasa Previa (Painless, rapid fetal deterioration)
1. RF: Placenta previa, Multiple gestations, IVF
2. BLEEDING OF FETAL ORIGIN → Rapid decline (total fetal BV ~250mL)
3. Fetal bradycardia (sinusoidal → severe anemia)
4. Management: C-section 34-35wks (before contractions/membrane rupture)

## Intrapartum Complications

### Shoulder Dystocia
- RF: Maternal obesity, GDM, Macrosomia >4.5kg, Postterm
- Dx: "Turtle sign" — fetal head retracts; Management: BE CALM
  - McRoberts (flex hips), Suprapubic pressure, Episiotomy
- Complications: Erb-Duchenne (C5-C7, "Waiter's tip"), Klumpke (C8-T1, "Claw hand" + Horner), Clavicle fracture

### Chorioamnionitis
- RF: Prolonged ROM >18hrs, PPROM, Prolonged labor, Internal monitoring, Repetitive vaginal exams
- Dx: Maternal fever + (Fetal tachycardia OR Leukocytosis OR Purulent fluid)
- Tx: BS Abx (Ampicillin + Gentamicin + Clindamycin) + DELIVERY; CI: Tocolytics

### Umbilical Cord Prolapse
- RF: PPROM, Malpresentation, Unengaged presenting part
- FHR: Recurrent variable decelerations; Tx: Maternal repositioning, Amnioinfusion

### Uterine Rupture (Painful)
- RF: Prior Classical C-section, Induction/prolonged labor, Macrosomia
- FHR: Abrupt abnormality (bradycardia, late/variable decels)
- CF: Sudden abdominal pain, SUDDEN LOSS OF FETAL STATION, Palpable fetal parts in abdomen
- Contractions: Staircase sign (progressively decreasing amplitude)
- Management: Emergency laparotomy + C-section

## Postpartum Complications

### PPH (>500mL vaginal or >1000mL C-section)

#### Uterine Atony (#1 cause)
- RF: Macrosomia, Prolonged induction, Operative delivery, Retained placenta, Grand multiparity
- PE: Soft, boggy uterus with fundus ABOVE umbilicus
- Tx: Bimanual massage → High-dose Oxytocin → Misoprostol (PGE1) → Tranexamic acid → Carboprost (CI: Asthma) → Methylergonovine (CI: HTN) → Balloon tamponade

#### Genital Tract Trauma
- RF: Operative vaginal delivery; CF: Laceration, vaginal sidewall defect; Tx: Repair

#### Retained Placenta
- Placenta Accreta: RF: Previa + Prior C-section/D&C; Dx: Numerous placental lacunae on US; Tx: Cesarean hysterectomy
- Retained POC: RF: Intraamniotic infection; Tx: D&C

#### Uterine Inversion
- RF: Macrosomia, Rapid delivery, Placenta accreta
- CF: Firm round mass through vagina, fundus NOT palpable; Hemorrhagic + Neurogenic shock
- Tx: Manual replacement → Laparotomy; Relaxants (Terbutaline, Nitroglycerin) if manual fails

### Late PPH (>24hrs)
1. Retained POC: Heavy bleeding ± Atony; Tx: D&C
2. Placental Site Subinvolution: Heavy bleeding + Atony; Tx: Uterotonics
3. Postpartum Endometritis (MCC postpartum fever): RF: C-section; Tx: Clindamycin + Gentamicin

### Other Postpartum
- Sheehan Syndrome: Obstetric hemorrhage → Pituitary infarction → Lactation failure, Amenorrhea, Fatigue, Hypotension
- Postpartum Thyroiditis: Lymphocytic infiltration; Hyperthyroid then Hypothyroid; Self-limited; Tx: BB
- Amniotic Fluid Embolism: Dyspnea, HYPOTENSION, DIC, Seizures; Tx: Respiratory/hemodynamic support
- Aspiration Pneumonitis: RF: Inc progesterone delays gastric emptying; Acute dyspnea, Low-grade fever post-aspiration
- Peripartum Cardiomyopathy: Last month gestation to 6 months postpartum

### Breastfeeding
- Contraindicated: Active TB, HIV (developed world), Herpetic breast lesions, Chemo, Cocaine/PCP
- Infant: Galactosemia
- Lactation problems: Engorgement → Plugged duct → Mastitis (Fever + Tx: Dicloxacillin) → Abscess (Fluctuant + Tx: US-guided drainage)
