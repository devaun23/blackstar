## Common Endo Diseases

- MC pituitary adenoma? -> Prolactinoma. Consider in amenorrhea/hypogonadism. Tx? Bromocriptine or cabergoline... even if macro (>10mm)
- Order of hormones lost in hypopituitarism? -> #1 FSH and LH, #2 GH, #3 TSH, #4 ACTH
- Polyuria, polydipsia, hyperNa, hyperOsm, dilute urine -> DI (lack of ADH or non-functional). Do water deprivation test to tell if crazy.
- Central DI -> urine Osm still decreased s/p water deprivation. Urine Osm increases w/ ddAVP
- Nephrogenic DI -> Urine Osm still decreased s/p ddAVP. Tx w/ HCTZ/amiloride.

## Hyperthyroidism

- See low TSH, high free T3/T4. Next best step? -> I-123 RAIU scan. If increased = Graves. If decreased = factitious or thyroiditis
- Tx? -> 1st = propranolol + PTU/MTZ. I-131 ablation or surgery (preggos & kiddos)
- Tx of thyroid storm? -> PTU + Iodine (Lugol's solution) + propranolol

## Work Up of a Thyroid Nodule

- 1st step? -> Check TSH
- If low? -> Do RAIU to find the "hot nodule". Excise or radioactive I-131
- If normal? -> FNA
- If benign? -> Leave it alone
- If indeterminate? -> Re-biopsy or check RAIU. Surgically excise and check pathology
- If malignant? -> Surgically excise and check pathology
- If cold? -> Surgically excise and check pathology

## Thyroid Cancer

- Papillary -> MC type, spreads via lymph, psammoma bodies
- Follicular -> Spreads via blood, must surgically excise whole thyroid!
- Medullary -> Assoc w/ MEN2 (look for pheo, hyperCa). Amyloid/calcitonin
- Anaplastic -> 80% mortality in 1st year
- Thyroid Lymphoma -> Hashimoto's predisposes to it

## Adrenal Issues — Cushing's

- Osteoporosis, central fat, DM, hirsutism -> Suspect Cushing's
- Best screening tests? -> 1mg overnight dexamethasone suppression test or 24hr urine cortisol
- If abnormal? -> Diagnoses Cushing's Syndrome
- Next best test? -> 8mg overnight dexamethasone suppression test
- Suppression to <50% of control? -> Pituitary adenoma (Cushing's disease)
- No suppression? -> Either adrenal neoplasia or ectopic ACTH
- Next best test? -> Plasma ACTH. Chest CT if smoker. Abdominal CT/DHEAS

## Adrenal Issues — Addison's

- Weakness, hypotension, weight loss, hyperpigmentation, increased K, decreased Na, decreased pH -> Suspect Adrenal Insufficiency
- Best screening test? -> Cosyntropin stimulation test (60min after 250mcg)
- MC cause? -> Autoimmune (Addison's disease)
- Treatment? -> NaCl resuscitation. Long term replacement of dexamethasone and fludrocortisone.

## Work Up of an Adrenal Nodule

- Best 1st step? -> Check functional status

| Diagnosis | Features | Biochemical Tests |
|-----------|----------|-------------------|
| Pheochromocytoma | High blood pressure, symptoms | Urine and plasma-free catecholamine metanephrines |
| Primary aldosteronism | High blood pressure, low K+, low PRA | Plasma aldosterone-to-renin ratio |
| Adrenocortical carcinoma | Virilization or feminization | Urine 17-ketosteroids |
| Cushing or "silent" Cushing | Cushing symptoms or normal exam | Overnight 1-mg dexamethasone test |

- If <5cm and non-functional -> Observe w/ CT scans q6mo
- If >6cm or functional -> Surgical excision

## Parathyroid Disease

- Hypoparathyroidism -> Perioral numbness, Chvostek, Trousseau s/p Thyroidectomy. Decreased Ca, increased PO4, decreased PTH
- Hyperparathyroidism -> Kidney stones, constipation/abd pain or psychiatric sxs. Increased Ca, decreased PO4, increased vitD, increased PTH. Dx w/ FNA of suspicious nodules. Can use Sestamibi scan. Tx w/ surgical removal of adenoma. If hyperplasia, remove all 4 glands and implant 1 in forearm.

## MEN Syndromes

- MEN1 -> pituitary adenoma, parathyroid hyperplasia, pancreatic islet cell tumor
- MEN2a -> parathyroid hyperplasia, medullary thyroid cancer, pheochromocytoma
- MEN2b -> medullary thyroid cancer, pheochromocytoma, Marfanoid

## Diabetes

- Diagnosis of Diabetes? -> FBGL > 126 x 2, 2hr OGTT > 200, random glucose > 200 + sxs (polyuria, polydipsia, blurred vision)
- Nausea, vomiting, abdominal pain, Kussmaul respirations, coma w/ BGL = 400? -> DKA. Dx? Ketones in blood (&urine), AGMA, hyperkalemia. Tx? High volume NS + insulin bolus & drip. Add K once peeing. Add glucose when <200
- Polyuria, polydipsia, profound dehydration, confusion and coma w/ BGL = 1000? -> HHS. Tx? High volume fluid & electrolytes. May require insulin.
- MC cause of death? -> Cardiovascular disease

## Diabetic Screening

- Heart? -> LDL < 100, BP < 130/80
- Kidney? -> Check for microalbuminuria (30-300 in 24hrs). Start ACE-I
- Eye? -> Annual screening for proliferative retinopathy -> Vitreous hemorrhage/neovascularization
- Nerves? -> Podiatric exam annually. Tx gastroparesis w/ metoclopramide or erythromycin. May get ED. 3rd, 4th, 6th CN palsy.
