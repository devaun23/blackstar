# Head Trauma

- GCS: eyes 4, motor 6, verbal 5
- Hematoma, edema, tumor can cause increased ICP
  - Symptoms: Headache, vomiting, altered mental status
  - Treatment: Elevate HOB, hyperventilate to pCO2 28-32, give mannitol (watch renal fxn)
  - Surgical intervention: Ventriculostomy
- **Epidural**: Lucid interval
- **Acute subdural**: Immediate LOC
- **Chronic subdural**: Elderly, EtOH

# Neck Trauma — Penetrating (GSW or Stab Wound)

- **Zone 3** = ↑ angle of mandible → w/u: Aortography and triple endoscopy
- **Zone 2** = angle of mandible to cricoid → w/u: 2D doppler +/- exploratory surgery
- **Zone 1** = ↓ cricoid → w/u: Aortography

# Penetrating Abdominal Trauma

- GSW to the abdomen → **Ex-lap** (plus tetanus prophylaxis)
- Evisceration → **Ex-lap** (plus tetanus prophylaxis)
- Stab wound & pt unstable, with rebound tenderness & rigidity → **Ex-lap** (plus tetanus prophylaxis)
- Stab wound but pt is stable → **FAST exam**. DPL if FAST is equivocal. Ex-lap if either positive.
- Blunt abdominal trauma with hypotension/tachycardia → **Ex-lap**

# Blunt Abdominal Trauma

- If unstable → **Ex-lap**
- If stable → **Abdominal CT**
  - Lower rib fx + bleeding into abdomen → Spleen or liver lac (Best test: Abdominal CT)
  - Lower rib fx + hematuria → Kidney lac
  - Kehr sign & viscera in thorax on CXR → Diaphragm rupture
  - Handlebar sign → Pancreatic rupture
  - Stable w/ epigastric pain → Best test: Abdominal CT. If retroperitoneal fluid → Consider duodenal rupture

# Pelvic Trauma

- If hypotensive, tachycardic → FAST and DPL to r/o bleeding in abdominal cavity
- Can bleed out into pelvis → stop bleeding by fixing fx (internal if stable, external if not)
- Blood at the urethral meatus and a high riding prostate → Consider pelvic fracture w/ urethral or bladder injury
  - Next best test: **Retrograde urethrogram (NOT FOLEY!)**
  - If normal → Retrograde cystogram to evaluate bladder
  - Looking for extravasation of dye. Take 2 views to ID trigone injury
  - Extraperitoneal extravasation → Bed rest + foley
  - Intraperitoneal extravasation → Ex-lap and surgical repair
