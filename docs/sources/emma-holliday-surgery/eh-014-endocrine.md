# Workup of a Thyroid Nodule

1. 1st step: **Check TSH**
2. If low → Do RAIU to find the "hot nodule". Excise or radioactive I131
3. If normal → **FNA**
   - If benign → Leave it alone
   - If indeterminate → Re-biopsy or check RAIU → Surgically excise and check pathology
   - If malignant → Surgically excise and check pathology
4. If cold → Surgically excise and check pathology

## Thyroid Cancer Types

- **Papillary**: MC type, spreads via lymph, psammoma bodies
- **Follicular**: Spreads via blood, must surgically excise whole thyroid!
- **Medullary**: Assoc w/ MEN II (look for pheo, hyperCa). Amyloid/calcitonin
- **Anaplastic**: 80% mortality in 1st year
- **Thyroid Lymphoma**: Hashimoto's predisposes to it

# Workup of an Adrenal Nodule

## #1: Check functional status

| Diagnosis | Features | Biochemical Tests |
|-----------|----------|-------------------|
| Pheochromocytoma | High BP, symptoms | Urine- and plasma-free catecholamine metanephrines |
| Primary aldosteronism | High BP, low K+, low PRA | Plasma aldosterone-to-renin ratio |
| Adrenocortical carcinoma | Virilization or feminization | Urine 17-ketosteroids |
| Cushing or "silent" Cushing syndrome | Cushing symptoms or normal exam | Overnight 1-mg dexamethasone test |

## #2: Size-based management

- If <5cm and non-functional → observe w/ CT scans q6mo
- If >6cm or functional → surgical excision

# Parathyroid Disease

## Hypoparathyroidism

- Typically comes from thyroidectomy
- Sxs: perioral numbness, Chvostek, Trousseau
- Labs: ↓Ca, ↑PO4, ↓PTH

## Hyperparathyroidism

- Usually asymptomatic ↑Ca, but can present w/ kidney stones, abdominal or psychiatric sxs
- Labs: ↑Ca, ↓PO4, ↑vitD, ↑PTH
- Dx w/ FNA of suspicious nodules. Can use Sestamibi scan
- Tx w/ surgical removal of adenoma. If hyperplasia, remove all 4 glands and implant 1 in forearm

# MEN Syndromes

- **MEN 1**: pituitary adenoma, parathyroid hyperplasia, pancreatic islet cell tumor
- **MEN 2a**: parathyroid hyperplasia, medullary thyroid cancer, pheochromocytoma
- **MEN 2b**: medullary thyroid cancer, pheochromocytoma, Marfanoid habitus
