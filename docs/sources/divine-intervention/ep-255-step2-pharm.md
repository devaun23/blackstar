# ANTIPLATELET AGENTS
Aspirin: stroke/carotid stenosis (1st line), acute MI, Kawasaki (+IVIG), preeclampsia ppx
Clopidogrel: 2nd line for stroke; P2Y12 receptor blocker
Dipyridamole: 3rd line; PDE3 inhibitor (increases cAMP → vasodilation)
Cilostazol: PAD (after supervised exercise program); PDE3 inhibitor
GpIIb/IIIa inhibitors: abciximab, eptifibatide, tirofiban

# PRIMARY HEMOSTASIS
Adhesion: GP1B-VWF → ADP release → P2Y12 activation → Aggregation: GpIIb/IIIa crosslinking

# ANTICOAGULANTS
## Warfarin
Indications: prosthetic valve (INR 2.5-3.5), valvular AFib (mitral stenosis)
MOA: inhibits Vit K epoxide reductase (factors 2,7,9,10 + protein C,S)
Bridge with heparin first (Protein C/S have shorter half-life → temporary hypercoagulable state)
INR 6-7: stop warfarin + oral vitamin K; Bleeding: reverse with 4-factor PCC
C/I: pregnancy (duodenal hematoma, rectus sheath hematoma); never give tPA with warfarin

## Heparin
MOA: activates AT-III → inhibits factors X and II; Reversal: protamine sulfate
## LMWH
Activates AT-III → inhibits factor X only
## Factor Xa inhibitors (-xaban)
## Direct thrombin inhibitors
Argatroban, dabigatran, bivalirudin; Indication: HIT; Dabigatran reversal: idarucizumab
## HIT
Platelets drop on heparin → stop heparin (don't give LMWH) → start direct thrombin inhibitor

# CARDIAC PHARMACOLOGY
## ACE-I
HTN in diabetics, CHF survival, nephroprotection, scleroderma renal crisis
C/I: bilateral renal artery stenosis, pregnancy (renal dysgenesis, oligohydramnios)
## Spironolactone
CHF survival, Conn syndrome, hirsutism in PCOS, portal HTN
## Beta-blockers
HOCM, CHF survival, migraines, akathisia, thyroid storm (propranolol)
C/I: acute cocaine intoxication; Overdose: atropine then glucagon
## Digoxin
Na-K ATPase blocker; hypokalemia predisposes toxicity; C/I: WPW
Causes of DCM: trastuzumab, doxorubicin, clozapine

# ANTIHYPERTENSIVES
Labetalol: mixed alpha/beta; safe in pregnancy (along with hydralazine, methyldopa, nifedipine)
Alpha-1 blockers: prazosin (PTSD nightmares), tamsulosin (BPH); AE: orthostatic hypotension
Alpha-2 agonist: clonidine (rebound HTN); 3rd line ADHD, Tourette's, opioid withdrawal
## Diuretics
Loops: lose calcium, ototoxic, like Bartter's syndrome
Thiazides: hypercalcemia, hyponatremia, like Gitelman's syndrome
## CCBs
Nicardipine/clevidipine: hypertensive emergencies
Nimodipine: post-SAH vasospasm
Nifedipine: pregnancy
Verapamil: can cause hyperprolactinemia
## Amiodarone
Class III (K channel blocker); V-tach; pulmonary toxicity
## Other
Procainamide: Class Ia, WPW, drug-induced lupus
Cardiac resynchronization: EF <35% + maximal medical therapy
Ivabradine: funny channel inhibitor → angina
