# QUALITY TERMINOLOGY
Quality Control: check EVERY instance (e.g., resident checks every med student note)
Quality Assurance: periodic audit (e.g., weekly note checks)
Quality Improvement: identified problem → designed intervention

# KEY EFFECTS
Hawthorne Effect: people act differently when observed
Weber Effect: tracking adverse events → incidence goes down (first year)
Safety Champion: person in charge of quality & safety

# HIGH VALUE CARE
Value = service x quality / cost

# QUALITY IMPROVEMENT MODELS

## PDSA Cycle
P = Plan, D = Do, S = Study, A = Act

## Lean Model
Find inefficiencies → eliminate them. Goal: reduce waste

## DMAIC (Data-driven improvement)
D = Define, M = Measure, A = Analyze, I = Improve, C = Control

## Six Sigma
No more than 3 defects per million products (6 standard deviations). Goal: eliminate defects

## FMEA (Failure Mode & Effects Analysis)
PROSPECTIVE TOOL: identify failure modes, rank them, focus on biggest error

## Root Cause Analysis
Like FMEA but RETROSPECTIVE TOOL

# SWISS CHEESE MODEL
Multiple safeguards must all fail (holes align) for error to reach patient
Example: wrong dose → EMR flags → pharmacist catches → nurse verifies
