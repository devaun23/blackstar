# BILIRUBIN METABOLISM
RBC → hemoglobin → heme + heme oxygenase → biliverdin → biliverdin reductase → indirect bilirubin
Indirect bili carried by albumin → liver → UDPGT conjugates → bile canaliculi → intrahepatic ducts → extrahepatic ducts → sphincter of Oddi → GI (some reabsorbed in terminal ileum)

# UNCONJUGATED/INDIRECT HYPERBILIRUBINEMIA

## Hemolytic causes
Autoimmune hemolytic anemia (type 2 HSR, lupus)
PNH (PIGA mutation → no GPI anchors → no CD55/CD59 → complement-mediated hemolysis)
Sickle cell, Thalassemia
G6PD deficiency (X-linked recessive; triggers: TMP-SMX, oxidants; women NOT affected)
Hereditary spherocytosis (AD; spectrin/ankyrin mutation; Dx: eosin-5-maleimide, osmotic fragility; Tx: splenectomy)

## Enzyme deficiency
Crigler-Najjar: UDPGT deficiency
Gilbert syndrome: mild UDPGT insufficiency → stressed → indirect hyperbili (benign)

## Neonatal jaundice
Physiologic: high Hct at birth (57%) + immature UDPGT → resolves
Rh/ABO incompatibility
Breastfeeding jaundice (1st week): insufficient feeding frequency
Breast milk jaundice (2nd week): substance in breast milk may inhibit UDPGT

# CONJUGATED/DIRECT HYPERBILIRUBINEMIA

## Neonatal
Biliary atresia: fibrosis of bile ducts → Kasai procedure in first days (>4 weeks → liver transplant)
DDx: choledochal cyst

## Pediatric
Asymptomatic: Dubin-Johnson (liver biopsy → black) vs Rotor syndrome (not black)
Pathophys: MRP1 transporter mutation

## Adult
PBC: intrahepatic bile ducts; anti-mitochondrial Ab; Tx: ursodiol
PSC: intra + extrahepatic ducts; p-ANCA+; beads on string; assoc: UC; Tx: ERCP for strictures
Choledocholithiasis: acute jaundice + mild RUQ pain (no fever)
Ascending cholangitis: + fever (usually from untreated choledocholithiasis)
Post-cholecystectomy + peritonitis → bile leak peritonitis (Dx: HIDA scan)
Pancreatic cancer: double duct sign on U/S
Cholangiocarcinoma

## Lab patterns
Cholestatic: direct bili elevated, alk phos markedly elevated >> AST/ALT
Hepatocellular: AST/ALT disproportionately elevated >> alk phos

## Alk phos conundrum
Alk phos elevated + GGT normal → bone problem
Alk phos elevated + GGT elevated → biliary etiology
5' nucleotidase: if elevated → liver problem
