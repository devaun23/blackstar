# B1 = THIAMINE
Cofactor for pyruvate dehydrogenase, alpha-ketoglutarate dehydrogenase, branching ketoacid dehydrogenase, transketolase
Deficiency population: alcoholics, hyperemesis gravidarum, eating disorders, starvation

## Wernicke's encephalopathy
Triad: Confusion + ophthalmoplegia + ataxia
Neuroanatomy: hemorrhagic infarction of mammillary bodies
Reversible! Tx: Give thiamine BEFORE glucose

## Korsakoff syndrome
Confabulation + amnesia → permanent

## Other presentations
Wet beriberi: generalized edema + CHF
Dry beriberi: ataxia + paralysis + sensory sxs

# B2 = RIBOFLAVIN
Required for FADH2 production

# B3 = NIACIN
Required for NADH/NADPH
Deficiency → Pellagra: dermatitis + diarrhea + dementia
Causes: Hartnup disease (can't reabsorb tryptophan), Carcinoid syndrome (tryptophan shunted to serotonin)
Therapeutic use: best way to raise HDL; AE: flushing + itching (Tx: NSAIDs)

# B5 = PANTOTHENIC ACID
Used to make coenzyme A

# B6 = PYRIDOXINE
Cofactor for transaminases, glutamate decarboxylase (glutamate → GABA), ALAS (1st step heme synthesis)
Deficiency → sideroblastic anemia, seizures
Drug cause: Isoniazid (also causes drug-induced lupus)

# B7 = BIOTIN
Cofactor for carboxylase enzymes; very rare deficiency
Odd cause: egg whites contain avidin

# B9 = FOLATE
Necessary for DNA synthesis; converts homocysteine → methionine
Causes of deficiency: alcoholism, small bowel disorders, poor nutrition, chronic hemolytic anemia
Drugs: phenytoin, methotrexate (rescue: leucovorin), TMP-SMX
Deficiency → megaloblastic anemia, neural tube defects
Lab: elevated homocysteine

# B12 = COBALAMIN
Converts homocysteine → methionine; converts methylmalonyl-CoA → succinyl-CoA
Deficiency → megaloblastic anemia, dementia, subacute combined degeneration (dorsal columns + lateral corticospinal tract)
Causes: pernicious anemia, Crohn's (terminal ileum), strict vegan diet, Diphyllobothrium latum
Lab: elevated homocysteine AND MMA

# VITAMIN C
Cofactor for collagen synthesis
Deficiency → Scurvy (bleeding gums + poor wound healing)
Therapeutic use: methemoglobinemia treatment (keeps iron in Fe2+)
