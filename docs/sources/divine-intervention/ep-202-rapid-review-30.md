# Congenital Infections

NB with microcephaly + seizures + facial defects, mom traveled to Brazil → Zika
Imaging: thin cerebral cortex + diffuse calcifications

## Congenital Infection Comparison
CMV: periventricular calcifications, sensorineural hearing loss, blueberry muffin rash
Toxoplasmosis: hydrocephalus, chorioretinitis, brain calcifications
Syphilis: snuffles
HIV: recurrent infections, chronic diarrhea
Zika: thin cerebral cortex, diffuse calcifications

# Pure Motor Stroke
Hx DM/HTN/smoking + 1/5 strength UE and LE + no sensory abnormalities
Location: posterior limb of internal capsule
Pathophys: rupture of Charcot-Bouchard microaneurysm in lenticulostriate arteries
Contrast MCA stroke: mostly arm & face (not LEs)

# Transfusion Reactions

## Acute Hemolytic
Systems error (Type B → Type A), Flank pain + hematuria
Pathophys: Type 2 HSR, pre-formed Ab
Dx: Direct Coombs test

## IgA Deficiency Anaphylaxis
Stridor + airway compromise
Pathophys: Type 1 HSR — 1st transfusion → Ab to IgA → class switching to IgE via IL-4 → 2nd transfusion → mast cell degranulation
Tx: epinephrine

## Febrile Non-Hemolytic
Fevers + chills 1-6 hrs after transfusion
Pathophys: Cytokines from WBCs in donor blood
Tx: Tylenol

## Delayed Hemolytic
Hematuria/mild sxs DAYS after transfusion + positive Coombs
Pathophys: Previous exposure to minor antigens (Kidd, Duffy) → memory B cells produce Ab on re-exposure

## TRALI
SOB + crackles + diffuse infiltrates, PCWP NORMAL
Pathophys: Anti-leukocyte Ab → inflammatory response (ARDS)

## TACO
SOB + crackles + diffuse infiltrates, PCWP ELEVATED (or CVP elevated, BNP elevated, new S3)
Pathophys: Cardiogenic pulmonary edema

## Dilutional Thrombocytopenia
Bleeding after large-volume transfusion (platelets not replaced)

# Trinucleotide Repeat Diseases

## Huntington's (CAG, AD, anticipation)
42 yo + involuntary movements (chorea) + behavioral changes, father died at 50
Tx of movement: Tetrabenazine

## Myotonic Dystrophy (CTG, AD)
23 yo + can't release handshake + bald → "cataracts, toupee, gonadal atrophy"
Gene: DMPK — Can present as NB hypotonia

## Friedreich's Ataxia (GAA, AR)
Child + ataxia + scoliosis + pes cavus → "GAAIT"
Cause of death: Hypertrophic cardiomyopathy

## Fragile X (CGG, X-linked dominant)
Boy + ID/autism + long face + big ears + macroorchidism + MVP → "chin giant gonads"
Gene: FMR1 — Pathophys: CGG repeats → methylation → silencing
Other X-linked dominant: Alport ("can't see, can't pee, can't hear a bee")

# Rescue Agents

## Acetaminophen → N-acetylcysteine
NAPQI → hepatocellular necrosis. NAC replenishes glutathione.

## Nitroprusside (cyanide toxicity) → Amyl nitrate + sodium thiosulfate
Cyanide impairs complex IV → anaerobic metabolism → lactic acidosis
Fe2+ doesn't bind cyanide, Fe3+ does → induce methemoglobinemia → Fe3+ binds cyanide → thiosulfate forms thiocyanate → renal excretion
Alternative: Hydroxocobalamin (B12 derivative)

## Iron poisoning → Deferoxamine
Baby eats mom's medicine → bloody stools + emesis + lactic acidosis
Dx: Babygram (radiopaque pills)

## Opioid → Naloxone

## Methemoglobinemia → Methylene blue
Fe3+ can't bind O2
Associated drugs: Dapsone, Nitrates, Lidocaine, Primaquine

## Carbon monoxide → Hyperbaric oxygen

## Lithium → Dialysis (if seizures)
