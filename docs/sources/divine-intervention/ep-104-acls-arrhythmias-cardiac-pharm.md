# Code Blue Protocol

Who is running the code?
- Identify yourself if running the code
- Assign tasks: Electricity/defib/cardiovert, Recorder, Timer, Pharmacy, Pill-pusher

Patient codes in front of you:
1. Tap the patient: do they respond?
2. Feel for a pulse (carotids)
3. Start CPR (30 compressions 120/min : 2 breaths)
4. Call for help
5. Try to identify arrhythmia on monitor


# VFib / Pulseless VTach Algorithm

Defibrillate = unsynchronized cardioversion 200 joules q2m
- Pulse and rhythm check prior to defibrillation
- Cycle: Epinephrine 1mg -> 2m -> Amiodarone 300mg IV -> 2m -> Epinephrine 1mg -> 2m -> Amiodarone 150mg IV -> 2m -> Epinephrine 1mg -> 2m -> Amiodarone 150mg IV -> 2m -> Lidocaine 1mg/kg
- "Every 2 minutes, do 3 things: pulse/rhythm check + shock + drug"


# PEA / Asystole Algorithm

No pulse = pulseless electrical activity or flat-line = asystole
- Epinephrine 1mg q4m
- "All roads lead to epi"
- Continue CPR 30:2
- DDx 5Ts: tension pneumo, thromboses, toxins, trauma, tamponade
- DDx 5Hs: hypo/hyperkalemia, hypothermia, hypoglycemia, H+/acidosis, hypovolemia


# Wide-Complex Tachyarrhythmia

Assume VTach. Pulse?
- No pulse: ACLS algorithm (defibrillate)
- Yes pulse: stable or unstable (hypotensive, altered, symptomatic)?
  - Stable: amiodarone
  - Unstable: synchronized cardioversion 100 joules (don't defibrillate during repolarization)


# Narrow-Complex Tachyarrhythmia (SVT)

1. Vagal maneuvers, cold water, massage carotids
2. Adenosine 6 -> 12 -> 12mg (trying to slow heart down)
3. IV beta blocker or CCB (non-dihydro: verapamil, diltiazem)
4. Synchronized cardioversion


# AFib

Irregularly irregular w/o p-waves
- Stable: Rate control (beta blocker, non-dihydro CCB), Rhythm control (amiodarone class III)
- Unstable: Synchronized cardioversion!


# Atrial Flutter

Sawtooth pattern
- Stable: Rate control (beta blocker, non-dihydro CCB), Rhythm control (amiodarone)
- Unstable: Synchronized cardioversion!


# WPW

Delta wave + short PR interval
- Do NOT block AV node (they'll keep running through bundle of Kent!)
- Give procainamide


# Multifocal Atrial Tachycardia

3+ p-wave morphologies
- Treat underlying disorder
- BB, CCB, stop smoking


# Bradyarrhythmia (HR in 30s)

1. Atropine (muscarinic antagonist -> speed up conduction through AV node)
2. Pacing


# Anti-Arrhythmic Drugs

## Class I: Sodium Channel Blockers (prolong QRS)

Class IA: Disopyramide, Quinidine, Procainamide
- Procainamide AE: drug-induced lupus
- Phenytoin also class IA

Class IB: Lidocaine, Mexiletine, Tocainide

Class IC: Flecainide, Propafenone

## Class II: Beta Blockers
Metoprolol, Labetalol

## Class III: K+ Channel Blockers
Amiodarone
- AE: hypotension, hypo/hyperthyroidism (Wolff-Chaikoff), blue discoloration of skin, pulmonary fibrosis (decreased DLCO)

## Class IV: Non-Dihydropyridine CCBs
Verapamil, Diltiazem
- Slow conduction through AV node at phase 4


# Digoxin

Mechanism: Positive inotrope + muscarinic agonist
- Inhibits Na/K ATPase pump -> increased intracellular Na -> inhibits Na/Ca exchanger -> increased Ca -> increased strength of contraction
- Does NOT improve survival!
- AE: digoxin toxicity (especially when hypokalemic), hyperkalemia (e.g. on diuretic)
- Tx of toxicity: anti-dig Fab fragment


# Heart Failure Drugs That Improve Survival

- Beta blockers: metoprolol, bisoprolol, carvedilol
- Spironolactone (aldosterone antagonist)
  - Also for PCOS, liver disease. AE: gynecomastia
- BiDil = hydralazine + isosorbide dinitrate
- ACE-I
  - C/I: bilateral renal artery stenosis
  - Especially useful for diabetics (protects efferent arteriole)
  - Also treats CCB-induced peripheral edema (dilates post-capillary venules)


# Other Cardiac Drugs

Nitroprusside: hypertensive emergency. AE: cyanide toxicity
Hydralazine: vasodilator. AE: drug-induced lupus, tachycardia
Cardiogenic shock:
- Dobutamine = beta-1 agonist -> increases HR, stroke volume
- Milrinone = inodilator/PDE inhibitor -> increases cAMP (at cardiac muscle: contraction; at smooth muscle: relaxation and decreased SVR)
