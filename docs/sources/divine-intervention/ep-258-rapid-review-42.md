# EPIGLOTTITIS
Any age + fever 102; Unvaccinated → H. flu; Vaccinated → S. pneumo
NBS: 1. Endotracheal intubation 2. Ceftriaxone/cefotaxime 3. Lateral neck XR (thumb sign)

# TUBERCULOSIS
Returned from deployment + chronic cough + hemoptysis + weight loss
Dx: BAL or induced sputum w/ AFB stain + culture; RIPE + B6
INH AE: peripheral neuropathy, seizures (B6 cofactor for glutamate decarboxylase), hepatotoxic, sideroblastic anemia, drug-induced lupus (anti-histone Ab)

## TB Induration
5mm: HIV; 10mm: HCW; 15mm: everyone

## Acid-fast organisms
HIV + diarrhea + acid-fast oocysts → Cryptosporidium (Tx: paromomycin, nitazoxanide)
Nocardia: also acid-fast

# LYME DISEASE
Northeast US + joint pain → Borrelia burgdorferi (spirochete)
Dx: ELISA + Western blot; Tx: <8 yo or pregnant → amoxicillin; >8 yo → doxycycline
Erythema chronicum migrans → clinical dx → doxycycline
CNS Lyme → ceftriaxone/cefotaxime
Jarisch-Herxheimer reaction: treating spirochetes → endotoxin release

# SBP
Ascites + abdominal pain + AMS → paracentesis >250 neutrophils → Tx: ceftriaxone/cefotaxime
Surgical antibiotic timing: cefazolin 30-60 min before incision; if time changed → re-dose

# CMV
Sore throat + joint pain + intranuclear inclusion → Monospot negative
Tx: ganciclovir; if resistant → foscarnet
Congenital CMV: periventricular calcifications
Congenital toxo: diffuse cortical/basal ganglia calcifications

## CALCIFICATION DDx
Frontal lobe + fried egg → oligodendroglioma
Suprasellar + bitemporal hemianopsia → craniopharyngioma
Posterior fossa + high Hct → hemangioblastoma
Around heart → constrictive pericarditis
Pancreas → chronic pancreatitis
Gallbladder (porcelain) → cholangiocarcinoma risk → elective cholecystectomy
Pouch of Douglas/adnexal → dermoid cyst
Kidney + HTN → primary hyperparathyroidism
DM + ABI >1.5 → Monckeberg sclerosis → Dx: toe-brachial index
