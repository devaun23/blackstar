# HSV ENCEPHALITIS

Young adult, AMS for last 3d, close acquaintances think they're "weird recently", obtunded, not an alcoholic, CSF with RBC: HSV encephalitis
Sx: encephalitis/meningitis in newborn or young adult
Dx: @ temporal lobe

## DDx
Subarachnoid hemorrhage: 2-3h duration, bloody CSF tap
C/b increased ICP → uncal herniation (transtentorial herniation) --| CN3 → "down and out" eye w/ blown pupil (intact CN4,6), --| midbrain/contralateral cerebral peduncle → ipsilateral hemiparesis

# TUBERCULOSIS

TSA agent, IVDU (#risky behavior), 2wks losing weight, night sweats, hemoptysis, CXR hilar adenopathy + cavitary lesion: TB
RF: travel, immunocompromised
Dx: serial acid-fast stains, TB skin test

## Induration
- >15mm: TB for all!
- >10mm: healthcare worker
- >5mm: HIV pt

f/u positive results with CXR → latent vs active TB
*negative skin test does not rule out TB! (i.e. sarcoid, HIV)
Enhancement @ base of brain
Elevated lymphocytes in CSF

## Treatment
Latent: isoniazid 9mo + B6/pyridoxine
AE: drug-induced lupus, slow vs fast acetylators
B6 deficiency --| glutamine decarboxylase → increased glutamate → seizures
B6 deficiency --| ALAS (heme pathway) → sideroblastic anemia

Active: RIPE 2mo + RI 4mo
AE:
- Rifampin: hepatotoxicity, orange secretions (also used to treat leprosy or for Neisseria meningitidis ppx)
- Isoniazid: B6 deficiency, neurotoxicity, hepatotoxicity; "INH"
- Pyrazinamide: n/a
- Ethambutol: color blindness

C/b Pott's disease (vertebral bodies)

# HIDRADENITIS SUPPURATIVA

Painful sores under axilla, chronic: hidradenitis suppurativa
Et: inflamed apocrine glands (vs. sebaceous glands @ acne)
Tx: surgery

# RHABDOMYOLYSIS

Obtunded, alcoholic/elderly, elevated Cr: rhabdomyolysis
RF: crush injury, intense exercise
Dx: blood on dipstick w/o RBC on microscopy
- Hypocalcemia → prolonged QT
- Hyperkalemia → 1. peaked T-waves 2. wide QRS 3. sine wave 4. asystole
Tx: calcium gluconate, insulin + glucose, albuterol, sodium bicarbonate, diuresis w/ furosemide, kayexalate
C/b acute tubular necrosis → Tx: fluids

Drug overdose, wide QRS: TCA

# HYPERSENSITIVITY REACTIONS

| Type | Mechanism | Examples |
|---|---|---|
| Type 1 | IgE (basophils, mast cells) crosslinking → release of cytokines | Asthma, ABPA (fungus ball in lung) |
| Type 2 | Antibodies against fixed antigen | AIHA (RBC), ITP (GpIIbIIIa), Graves disease (TSH receptor) |
| Type 3 | Antigen-antibody complex | PSGN, Hypersensitivity pneumonitis |
| Type 4 | Delayed-type; T-cells/macrophages | Contact dermatitis, TB skin test, Poison ivy |
