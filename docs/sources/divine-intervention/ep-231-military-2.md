# MILITARY VACCINES/PROTECTION
MMRV, Tdap, typhoid, influenza, polio, hep A + location-dependent (yellow fever, anthrax, smallpox)
DEET on skin, permethrin on clothes
Medic or sexually active → Hep B
Meningitis outbreak → ceftriaxone/ciprofloxacin + rifampin ppx for close contacts

# TROPICAL/MILITARY INFECTIONS

RSM from Egypt with hematuria or portal HTN → Schistosomiasis (haematobium = hematuria, mansoni = portal HTN)
MCC diarrhea in military → Norovirus
Peace keeping in Africa + bleeding + contact with wild animals/dead bodies → Ebola or Marburg (fomites/body fluids, no Rx)
RSM from Russia/China/Korea/Utah/NM/AZ/CO + contact with mice + flu-like + bilateral interstitial pulmonary edema → Hantavirus (no known treatment)
RSM from Hawaii/navy + conjunctival injection + jaundice → Leptospirosis (bad form = Weil's Disease — anemia, high Cr, jaundice; ppx: doxycycline)
RSM + hemoptysis + widened mediastinum ± necrotic ulcer → Anthrax (poly D-glutamate capsule; hemorrhagic mediastinitis; Rx: doxy or FQ; vaccine available)

## Malaria
PPX: atovaquone + proguanil, mefloquine, primaquine (for vivax/ovale — tertian fever, hypnozoite form), chloroquine (high resistance), doxycycline
P. malariae = quartan fever; P. falciparum = worst (cerebral malaria, pulmonary edema, renal failure)
Before primaquine → check for G6PD deficiency

RSM + animal/unpasteurized milk + undulant fevers + profuse sweating → Brucellosis (Rx: doxy + rifampin; if recurring: aminoglycosides)
Contact with dogs/cats/pigs + bloody diarrhea → Campylobacter jejuni (assoc: GBS, Reiter's)
Cattle/sheep birth + pneumonia or culture-negative endocarditis → Q fever (Coxiella burnetii; pneumonia differentiates from brucellosis)
RSM from tropical areas + fevers + pancytopenia + hepatosplenomegaly + dark skin lesions → Leishmaniasis (visceral/donovani; sandflies; Rx: sodium stibogluconate or amphotericin B)
RSM + fever + neck rigidity + LP: normal opening pressure, lymphocytes, not high protein/glucose → West Nile virus
RSM + achalasia + megacolon + S3 + periorbital edema (Romana sign) → Chagas disease (reduviid bug; Rx: nifurtimox, benznidazole)
RSM + turtles/eggs/poultry + bloody diarrhea + neutrophilia → Nontyphoidal Salmonella (DON'T TREAT)
RSM + chronic cough + hemoptysis + night sweats + upper lobe infiltrate → TB (RIPE + B6)
RSM + very high fevers + rash + bone-break fevers + bleeding + thrombocytopenia + positive tourniquet test → Dengue hemorrhagic fever (Aedes mosquito; increased capillary permeability)
Chikungunya = identical to dengue but without hemorrhage
Zika = woman with child who has microcephaly + intracranial calcifications
Yellow fever = jaundice + hepatomegaly (not present in dengue)
