# MENSTRUAL CYCLE PHYSIOLOGY

Pt with a 40-day long menstrual cycle. How long is the follicular phase?
Luteal phase is always 14 days, follicular phase is 26 days
Dominant follicle makes estrogen → proliferative endometrium
LH surge → ovulation
Corpus luteum makes progesterone → converts proliferative to secretory endometrium
If egg fertilized → zygote implants and makes beta-hCG → maintains corpus luteum
If not fertilized → corpus luteum degrades after 14 days (FIXED)

## Progestin withdrawal test
Supply progestin and withdraw it — person should bleed
If they get a period → progestin deficiency was the problem → didn't ovulate → Consider PCOS

# MLF VS PPRF LESIONS

## MLF (Medial Longitudinal Fasciculus)
Location? Pons
Function? Makes horizontal conjugate gaze work
Looking to R: R eye abducts (R CN6) + L eye adducts (L CN3)
MLF allows communication between CN6 & CN3
If L MLF lesion → R eye abducts but L is stuck
Impaired in INO (often MS pt)
MLF impaired = same side as eye that can't adduct — only 1 eye affected

### MLF vs CN3 lesion
MLF: cannot adduct during horizontal conjugate gaze but CAN adduct for accommodation (normal convergence)
CN3 lesion: cannot adduct for either (e.g. compression from PCOM aneurysm)

## PPRF
Location? Pons
Function? Drives CN6 to create horizontal conjugate gaze
If R PPRF lesion → L PPRF acts unopposed → eyes deviate to L
For PPRF lesion: both eyes deviate AWAY from side of lesion — both eyes affected

# HARTNUP DISEASE

Pt with Hartnup disease — most likely psych comorbidity? Depression
Pathophys? Defect in transporter for neutral amino acids → can't absorb tryptophan → can't make serotonin or niacin

# TRAUMA

## Pulmonary contusion
Pt in MVC with bilateral interstitial infiltrates + hypoxia

## Cardiac contusion
Pt in MVC with CP, elevated PCWP, mildly elevated troponins

## Blunt trauma to kidney
1st step? UA (look for hematuria) → if blood in urine → CT abdomen → if UA negative → stop there!

# ELEVATED ALKALINE PHOSPHATASE

Causes?
- Obstructive biliary process (choledocholithiasis, ascending cholangitis, PBC)
- Bone (Paget's disease of bone)
Check GGT — if GGT elevated too → biliary cause
