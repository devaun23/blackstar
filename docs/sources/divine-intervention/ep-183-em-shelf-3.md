# STDs

## Chlamydia
Penile discharge + no organisms on microscopy -> chlamydia
Sxs: Urethritis, Epididymitis (young pt -> CT/NG; old pt -> E. coli), Non-purulent discharge in women, PID (cervical motion/adnexal tenderness), Reiter's syndrome (conjunctivitis + urethritis + arthritis "Can't see, can't pee, can't climb a tree")
Dx? NAAT (male: urine; female: vaginal swab or urine)
Tx? Azithromycin (single dose) or doxy

## Gonorrhea
GS with gram-neg diplococci -> gonorrhea
Sxs: Urethritis, Epididymitis, Purulent discharge, PID, Petechial rash (disseminated)
Dx? NAAT. Tx? Ceftriaxone + azithro (single dose). Avoid sexual contact 7 days after abx

## Trichomonas
Severe vulvar tenderness + foul-smelling greenish thin discharge + pH > 4.5 -> trichomonas
Dx? Wet mount (motile organisms). Tx? Metronidazole (alternative: tinidazole)

## Syphilis
MSM + painless penile ulcer -> syphilis
- Primary: Chancre (painless ulcer w/ indurated borders)
- Secondary: Rash on palms & soles, condyloma lata
- Tertiary: Gummas, Tabes dorsalis (destruction of dorsal columns, positive Romberg)
- Neurosyphilis, Aortitis -> aortic dissection
- Congenital: Sniffles in NB
Dx: RPR/VDRL first -> FTA-Abs. Dark-field microscopy
Tx: Primary/secondary -> IM penicillin G (if PCN allergic: doxy). Tertiary -> 3 weeks penicillin G
Jarisch-Herxheimer reaction: Fevers, HAs, myalgias after PCN tx

## Genital Herpes
Multiple vesicles on erythematous base -> HSV-2
Painful genital ulcer DDx: Herpes, Chancroid (H. ducreyi)
Dx? PCR, Viral culture (NOT Tzanck smear). Tx? PO acyclovir/valacyclovir/famciclovir
Admit if very high fevers/not tolerating PO -> IV acyclovir. Resistant -> foscarnet

## Chancroid
Very painful papule + exquisitely tender + enlarged inguinal nodes -> H. ducreyi
Tx? Ceftriaxone or azithromycin (single dose)


# Toxic Shock Syndrome

24 yo F on tampons. Temp 103F, BP 70/40, rash, desquamation, elevated Cr -> TSS
Also: Nasal packing, SSTI
Bug? Staph aureus or Strep pyogenes. Pathophys? Superantigen
Tx: Source control (remove foreign body) -> IVF -> vasopressors -> Vanc + clindamycin (inhibits toxin production). If GAS -> pip-tazo or meropenem + clinda. Not improving -> IVIG


# Sepsis

SIRS criteria (2/4): T <36 or >38, RR >20, HR >90, WBC <4k or >12k
SIRS + source -> sepsis. Sepsis + organ dysfunction -> severe sepsis. Hypotensive despite volume resuscitation -> septic shock
Most likely pulmonary complication? ARDS

Management: IVF (2 L bolus) -> Norepinephrine (central line) -> Vasopressin -> Broad-spectrum abx ASAP
- Empiric: Vanc + carbapenem
- Pulmonary: Vanc + ceftriaxone + azithro + FQ
- Neutropenic: Cover pseudomonas (ceftazidime, cefepime, carbapenem, pip-tazo)
- Not improving on vanc + ceftazidime -> add antifungal (caspofungin)
Source control: Remove infected line, I&D abscess


# Skin & Soft Tissue Infections

## Cellulitis
Bugs: Staph (MRSA) + Strep. Dx? Clinical. Tx: MRSA -> Clinda or TMP-SMX. Strep -> cephalexin. Admit if systemic -> vancomycin

## Erysipelas
Fever + well-demarcated tense erythematous rash on face -> Strep pyogenes. Tx: PCN/amox/cephalexin. Systemic -> IV cefazolin

## Necrotizing Fasciitis
Looks like cellulitis + exquisitely tender + skin discoloration + bullae/crepitus + tachycardia -> necrotizing fasciitis
Imaging: Gas in soft tissue. Labs: Elevated CK (C. perfringens), elevated Cr, CBC abnormalities
Tx: Vanc + carbapenem + clindamycin + surgical debridement

## Abscess: Fluctuant mass -> I&D. Bartholin gland abscess -> drain + Ward catheter
## Hidradenitis Suppurativa: Painful masses in axillae. Tx: surgery
## Pilonidal Cyst: Painful mass in intergluteal fold. Tx: surgery
## Sporotrichosis: Gardener + subcutaneous nodules following lymph chain. Tx: Itraconazole for weeks


# Influenza

Abrupt fever/chills + HA + myalgias in December -> influenza. Kids: diarrhea
Secondary bacterial pneumonia: Improving then febrile again + consolidation -> MRSA. Aspirin c/i in kids (Reye syndrome). Neuro complication: GBS
Dx: Low-risk + classic sxs -> none. High-risk -> rapid antigen test (not sensitive, pretty specific)
Tx: Neuraminidase inhibitor within 48 hrs. PO -> oseltamivir. Inhaled -> zanamivir (c/i: airway disease). IV -> peramivir
Secondary bacterial pneumonia tx -> vancomycin or linezolid


# Herpes Manifestations

Oral, Genital, Herpetic whitlow, Skin (wrestlers), Herpes keratitis (dendritic lesion on slit lamp), HSV encephalitis (temporal lobe enhancement, LP with RBCs + lymphocytes, tx: IV acyclovir)

Chicken pox: Vesicular rash, lesions in different stages. Contagious until all crusted. Contrast smallpox: same stage, extremities, larger
Shingles: Dermatomal vesicular lesions. Tx: Acyclovir within 72 hrs


# EBV / Mono

Young F + sore throat + cervical LAD + exudative pharyngitis + splenomegaly -> mono
Dx? Monospot (if negative consider CMV). Associations: Hodgkin's lymphoma, Burkitt's (t(8,14), starry sky), Nasopharyngeal carcinoma
Histology: Atypical lymphocytes (Downey cells). Tx: Supportive + avoid contact sports 3-4 weeks. Airway compromise -> steroids


# Measles

Child with T 103F, runny nose, cough, conjunctival injection, Koplik spots, rash head-to-body -> measles
Management: Contact health department + Vit A


# Hemorrhagic Fevers

Recent travel to Africa + bleeding from ears & nose -> hemorrhagic fever (Ebola). NBS: Isolation + PPE. Tx: Supportive
Rodent exposure + fever + hypotension + AKI -> hantavirus hemorrhagic fever w/ renal syndrome


# HIV

MSM + 3 weeks low-grade fevers + sore throat + rash + generalized LAD -> HIV
Dx? Antigen (p24)/antibody testing. Tx? HAART (2 NRTIs + 1 other)

Opportunistic infections:
- Esophagitis -> candida (tx: fluconazole), CMV, HSV
- Thrush -> nystatin or fluconazole
- IRIS: Low CD4 treated then fevers/flu-like illness
- Retinitis -> CMV (tx: ganciclovir; resistant: foscarnet)
- Pneumonia: MC S. pneumo. PCP: profound hypoxia + increased LDH + interstitial infiltrates. Tx: IV TMP-SMX. Steroids if PaO2 <70 or A-a gradient >35
- Seizures + ring-enhancing lesions -> Toxo. Tx: Pyrimethamine-sulfadiazine
- High fevers + nuchal rigidity + high OP -> Cryptococcal meningitis. Dx: Latex agglutination. Tx: Amphotericin B + flucytosine -> 1 yr fluconazole
- Chronic watery diarrhea + acid-fast oocysts -> Cryptosporidium. Tx: Nitazoxanide
- Late-stage AIDS + diarrhea -> MAC or CMV. MAC tx: REC (rifabutin + ethambutol + clarithromycin)
- Brown-purple nodules -> Kaposi's sarcoma

HIV Prophylaxis:
- CD4 <250 + southwest US -> cocci ppx (itraconazole)
- CD4 <200 -> PCP (TMP-SMX)
- CD4 <150 + midwest -> histoplasma (itraconazole)
- CD4 <100 -> toxo (TMP-SMX)
- CD4 <50 -> MAC (azithromycin)

Needle stick PEP: Start within 2 hrs. PEP = HAART. High-risk: high viral load, deep injury, into blood vessel


# Endocarditis

IVDU + 2 weeks fever + new murmur -> endocarditis
Bugs: #1 S. aureus, #2 S. viridans, #3 Enterococci. New prosthetic valve (<60 days): S. epidermidis. Post-GI/GU procedure: Enterococcus
Valve: Tricuspid in IVDU, Bicuspid in others
Exam: Osler's nodes, Janeway lesions, Splinter hemorrhages
Dx: 3 blood cx from 3 sites (before abx) + TEE. Empiric: Vanc + gent +/- rifampin

Abx ppx (amoxicillin): Dental, GI/GU procedures with infection, Respiratory tract procedures, Procedures on infected skin/MSK
Who: Hx endocarditis, Prosthetic valve, Heart transplant with valve dysfunction, Unrepaired cyanotic congenital heart defect


# Tetanus

Puncture wound/animal bite/unsterile abortion + jaw stiffness + muscle spasms + rigid extremities -> tetanus
Pathophys: Tetanus toxin cleaves SNARE proteins -> can't release glycine & GABA -> spastic paralysis
Neonatal: Poor prenatal care + unsanitary delivery + poor feeding + clenched fists
Tx: Admit quiet room, Tetanus immune globulin + tetanus toxoid vaccine (different arms), Debride necrotic tissue, Benzos, If autonomic hyperactivity -> beta-blocker, Often intubate + paralyze (vecuronium/rocuronium)


# Rabies

Dog bite weeks ago -> hyperactive episodes + hallucinations + hypersalivation + hydrophobia -> coma -> rabies
Causes: Dog, bat, skunk, raccoon. Very poor prognosis. Dx: Postmortem brain pathology
PEP: Rabies immune globulin + rabies vaccine (2 sites). When: Any bat contact, stray animal, animal that can't be observed. Domestic animal -> quarantine 10 days


# Malaria

Missionary from Africa + cyclic fevers + splenomegaly + low Hgb + low platelets -> malaria
Vector: Anopheles mosquito. Dx: Thick & thin blood smear
- P. falciparum: Worst (cerebral malaria, jaundice, renal impairment)
- P. vivax/ovale: Hypnozoites in liver. Tx: Primaquine (c/i: G6PD deficiency)
- P. malariae
Tx: Artemether-lumefantrine, Atovaquone-proguanil, Quinidine + doxy (quinidine AE: QT prolongation)


# Other Infections

## Vibrio vulnificus: Longtime alcoholic + cellulitis + coastal swimming/shellfish. Tx: Doxy + cefepime + debridement
## Aeromonas: Freshwater exposure + cellulitis
## Pseudomonas folliculitis: Hot tub + swimsuit-shaped rash. Usually self-limited
## Hypersensitivity pneumonitis: Frequent hot tub use + respiratory sxs -> thermophilic actinomycete
## Mycobacterium marinum: Water worker + skin lesions + granulomas on biopsy


# Diarrhea Summary

Watery: Cruise ship -> norovirus. Kid -> rotavirus. Potato salad <6 hrs -> S. aureus. Profound -> V. cholera (tx: ORS). HIV -> cryptosporidium
Bloody: EHEC, Shigella, Campylobacter, Salmonella. Pseudoappendicitis -> Yersinia. Liver abscess -> Entamoeba histolytica
HUS: E. coli O157:H7 or Shigella (MAHA + thrombocytopenia + elevated Cr)
Post-abx -> C. diff (watery or bloody). Tx: Oral vancomycin
General diarrhea tx: Rehydrate, antiemetic, if must pick abx -> FQ. Abx contraindicated in HUS

## Food Poisoning
Exotic fish + flushing + HA + autonomic instability -> scombroid (histamine toxicity, tx: antihistamine)
Contrast ciguatera: HAs, myalgias, numbness, reversal of hot-cold sensation


# Zoonotic & Travel Infections

## Brucellosis: Dog breeder + undulant fever. Tx: Doxycycline
## Capnocytophaga: Dog bite + osteomyelitis. Tx: Amox-clav or clindamycin
## Chlamydia psittaci: Bird breeder + flu/pneumonia/endocarditis. Tx: Doxy or azithro
## Q Fever: Cattle/sheep/goat worker + fever/pneumonia/endocarditis. Tx: Doxycycline
## Ehrlichiosis: Tick (Lone star) + fever/HA/myalgias + anemia + thrombocytopenia. Tx: Doxy for everyone
## RMSF: Tick + high fevers + rash starting palms/soles. Geographic: North Carolina. Dx: Weil-Felix. Tx: Doxy (chloramphenicol for pregnant)
## Leptospirosis: Swimming in Hawaii + conjunctival injection + LAD + fevers. Dark-field microscopy. Tx: PCN or doxy
## Tularemia: Rabbit exposure + ulcer + regional LAD. Also: oculoglandular form. Bioterrorism agent. Tx: Doxy or aminoglycosides

## Lyme Disease
Bullseye rash. Vector: Ixodes tick. Geographic: Northeast US
- Stage 1: Erythema chronicum migrans. Tx: Age 8+ -> doxy. <8 -> amoxicillin
- Stage 2: Heart block, bilateral Bell's palsy, disseminated rash, arthralgias. Tx: Ceftriaxone/cefotaxime
Dx: ELISA (screening) -> Western blot (confirmatory)

## Pulmonary Anthrax
FBI/postal worker OR wool-sorting + hemoptysis + widened mediastinum -> pulmonary anthrax
Cutaneous: Macule -> ulcer -> black eschar (good prognosis). Spore-forming bacteria
Tx: Cipro + meropenem + linezolid

## Bubonic Plague: Necrotic draining lymph nodes. Yersinia pestis. Tx: Aminoglycoside

## Tropical Diseases
- Dengue: Severe bone pain + hepatosplenomegaly + fevers. Vector: Aedes mosquito. Avoid aspirin/NSAIDs
- Chikungunya: Dengue-like illness
- Zika: Fever + rash + conjunctivitis + joint pain. NB: microcephaly. Neuro: GBS
- Typhoid: Very high fever + severe abdominal pain + bradycardia + salmon-colored abdominal lesion. Bug: S. typhi. Tx: FQ (if severe + dexamethasone)
- Yellow fever: International travel + no vaccines + flu -> hemoptysis + black emesis + proteinuria. Flavivirus. Tx: Supportive
- Neurocysticercosis: Pork consumption + seizures. Tx: AED + albendazole +/- praziquantel + corticosteroids. Dx: Calcified lesions on NCCT
- African trypanosomiasis: Africa + painless ulcer + cyclic fevers + somnolence -> coma. Vector: Tsetse fly
- Chagas: Unilateral periorbital edema + painful swelling at bite. Vector: Reduviid bug. Complications: Dilated cardiomyopathy, Achalasia, Megacolon. Tx: Benznidazole, Nifurtimox
- Schistosomiasis: Egyptian + hematuria. Vector: Snail. Complication: Bladder cancer
- Entamoeba: Bloody diarrhea + elevated eos + liver abscess. Tx: Metronidazole


# Helminths

- Ascaris: Travel + dry cough + constipation + elevated eos. Loeffler's syndrome (pneumonitis). Large burden -> SBO. Tx: Albendazole
- Pinworm (Enterobius): Perianal itching in kid. Dx: Scotch tape test. Tx: Albendazole or pyrantel pamoate
- Hookworm: IDA + construction worker + bug burrowing into skin. Tx: Albendazole
- Trichinella spiralis: Myalgias + cysts in striated muscle. Tx: Albendazole
- Tapeworms: Diphyllobothrium latum -> B12 deficiency. T. saginata (beef). T. solium (pork). Tx: Praziquantel


# Transplant Complications

- Stem cell transplant + rash + diarrhea -> GVHD. Tx: Steroids
- Liver transplant + severe abdominal pain + peritonitis -> bile leak. NBS: Surgical consult. Dx: HIDA scan
- Kidney transplant + infection -> BK polyomavirus
- Heart transplant rejection: NBS: High-dose methylprednisolone. Bradyarrhythmia -> isoproterenol (B1/B2 agonist). Cardiogenic shock -> dobutamine or milrinone
