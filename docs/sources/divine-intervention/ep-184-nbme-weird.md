# Most Common Complications Of

- Cleft lip/palate -> chronic OM / speech problems
- BM transplant -> GVHD
- Pregnancy complication of antiphospholipid Ab -> recurring spontaneous abortions (first 20 wks = spontaneous abortion)
- Terminal disseminated malignancy -> cachexia
- AAA -> rupture
- Chronic HTN -> LVH
- First few hours of life in baby of diabetic mom -> hypoglycemia
- Meckel's diverticulum -> bleeding
- Diverticulosis -> diverticulitis
- Appendicitis -> periappendiceal abscess
- Cholelithiasis -> biliary colic
- Upper UTI -> hydronephrosis
- BPH -> obstructive uropathy
- DM (esp T1) -> insulin-induced hypoglycemia
- Infectious complication of chronic granulomatous disease -> pneumonia
- Systemic amyloidosis -> renal failure
- Left atrial thrombus -> embolization
- Ganglion cyst resection -> recurrence
- Acute pancreatitis within 1-2 weeks -> pancreatic pseudocyst
- Influenza -> primary influenza pneumonia or secondary bacterial superinfection
