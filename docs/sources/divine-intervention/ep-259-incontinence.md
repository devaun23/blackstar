# PHYSIOLOGY
Continence = balance between bladder contracting and urethra closing
Urethral pressure normally exceeds bladder pressure → keeps us continent

# STRESS INCONTINENCE
Presentation: leaking with cough/sneeze/valsalva
Cause: urethra lost support → urethral pressure drops below bladder pressure
RF: pregnancy, obesity, menopause (decreased estrogen → decreased pelvic floor tone), prostate surgery
Dx: Q-tip test (>30° rotation)
Tx: Kegel exercises → mid-urethral sling procedure

# URGE INCONTINENCE
Most common type in men (caused by BPH)
Pathophys: detrusor hyperactivity/overactive bladder → constant urge
Post-void residual <10 mL
Tx: anticholinergics — "On The Darn Smelly Toilet" (Oxybutynin, Trospium, Darifenacin, Solifenacin, Tolterodine)

# OVERFLOW INCONTINENCE
Post-void residual = 100s of mL
Pathophys: detrusor hypotonia/hypomobility/areflexia
Tx: cholinergic agents (bethanechol, carbachol, neostigmine)
Acute setting: catheter
