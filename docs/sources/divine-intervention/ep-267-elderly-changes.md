# BRAIN
Gets smaller → susceptible to subdural hematomas (bridging veins stretch/shear)
Cerebral blood flow decreases
Non-specific white matter changes → normal (if asymptomatic)
Working memory declines; mild MMSE defects ≠ Alzheimer's
ADL impairment or getting lost → NOT normal, must work up

# NERVOUS SYSTEM
Mild lens opacification normal (if good visual acuity) ≠ cataracts
Anosmia is normal; High-frequency hearing loss; Vestibular system declines

# CARDIOVASCULAR
Pulse pressure widens (systolic ↑, diastolic ↓)
HR decreases (fibrosis of conducting system) → CO decreases
Baroreflex impaired
Blood vessels less compliant → increased systolic BP
Heart gets SMALLER (fewer/smaller cardiac myocytes); big heart is NOT normal

# LUNGS
Pulmonary reserve and FEV1 decline; Smoking accelerates but quitting helps
Lungs lose elastic recoil; Mucus harder to clear

# GI
Mild malabsorption (villous atrophy); GERD (esophageal peristalsis declines)
Liver mass decreases → reduced drug metabolism

# HEMATOLOGY
Decreased bone marrow density → anemia
Less calcium/iron absorption → fractures/osteoporosis

# RENAL
Mild proteinuria normal; Creatinine clearance declines → Cr slowly rises

# ENDOCRINE
Androgen levels down → libido down (testicular/ovarian atrophy)
Estrogen down → osteoporosis, painful sex; Sperm count declines
PTH goes up → more osteoporosis; Insulin secretion up; NE and vasopressin up
Thyroid hormone and ACTH down → metabolic rate down
Should NOT be able to palpate ovaries in elderly women → think cancer if palpable

# SKIN
Epidermis/dermis thin (decreased fibroblast activity → less collagen)
