# LIVER INJURY PATTERNS

## Hepatocellular Pattern
Labs: ↑↑ AST/ALT >> AP/GGT
Etiologies: APAP toxicity, Alcoholic hepatitis, HAV
Tx: Liver transplant if MELD score >18 or Marjorie score >32

## Cholestatic Pattern
Labs: ↑↑ ALKP/GGT >> AST/ALT + Direct hyperbilirubinemia
Etiologies: Biliary tree pathology, Hemolytic anemia, Gilbert syndrome

Alcoholic + AST:ALT ratio > 2:1 + ↑PTT and ↑PT/INR → Alcoholic Liver Disease (Tx: steroids)
Jaundice + ↑ALKP/↑GGT + recent illness/surgery/fasting → Gilbert Syndrome (indirect hyperbili, benign)

# VIRAL HEPATITIS

## HAV
Very acute-onset + RUQ pain + jaundice + ↑AST/ALT (1000s) + trip to Mexico
Dx: Anti-HAV IgM
Tx: supportive
Prevention: HAV vaccine for RF (travel, IVDU, cirrhosis, hemophilia, MSM)

## HBV
Transmission: blood (needle stick), birthing, bonking
Serologies summary — see full table in notes
Tx: Needle-stick + unvaccinated → HBV vaccine + IVIG; Vaccinated → nothing
Adults → supportive; Chronic → entecavir/tenofovir or pegylated IFN-alpha
Associations: PAN, Membranous Nephropathy

## HCV
Dx: Anti-HCV Ab then HCV RNA
Tx: sofosbuvir or simeprevir (check for HBV first!)
RF: Born 1945-1965, blood transfusions, IVDU
Associations: Mixed cryoglobulinemia, PCT, Leukocytoclastic vasculitis, HCC
Screen: U/S q6 months for HCC

## Autoimmune hepatitis
↑AST/ALT + ANA+ + h/o autoimmune disease
Type 1: anti-Sm Ab; Type 2: anti-LKM Ab
Tx: steroids or azathioprine

## Autoimmune pancreatitis
↑IgG4 + sausage-shaped pancreas

# HEREDITARY HEMOCHROMATOSIS
65 yo M + ED + skin hyperpigmentation + new-onset DM + pseudogout
Pathophys: iron deposition → oxidation → destruction
Dx: Transferrin saturation; Tx: Phlebotomy
Inheritance: AD; Mutations: HFE gene (C282Y)

# NAFLD
Very obese + mildly ↑AST/ALT + T2DM or HLD
Tx: treat underlying etiology

# PBC VS PSC

| | PBC | PSC |
|---|---|---|
| Demographic | women 40-60 yo | young men with UC |
| Location | Intrahepatic bile ducts | Intra + Extrahepatic |
| Labs | ↑ALKP/GGT, anti-mitochondrial Ab | ↑ALKP/GGT |
| Imaging | U/S | U/S → MRCP/ERCP ("string of beads") |
| Tx | Ursodiol, Vit ADEK, liver transplant | MRCP/ERCP to dilate strictures |
| Follow-up | | U/S q6mo for HCC, colonoscopy at dx then q1-2y |

# COMPLICATIONS OF CIRRHOSIS
For all cirrhosis patients: U/S q6 months to screen for HCC

Hepatic encephalopathy: 2/2 NH4+ accumulation. Triggers: infection, GI bleed, TIPS. Tx: lactulose or rifaximin
Hepatopulmonary syndrome: SOB + high A-a gradient. Dx: echo
Portopulmonary hypertension: SOB + high RV pressure. Dx: echo
Hepatorenal syndrome: ↑SCr, splanchnic vasodilation → renal hypoperfusion. Labs like prerenal (FeNa <1%)
Ca/PO4 in cirrhosis: ↓Ca + ↓PO4 + ↑PTH (liver can't make 25-OH-vitD). Tx: calcitriol, bisphosphonate

## GI Bleed Management
Step 1: 2 large-bore IVs
Step 2: Fluids (+blood if Hb <7)
Step 3: EGD
Step 4: Colonoscopy
Step 5: Tagged RBC scan
+ IV octreotide + PPIs

## SBP
Dx: Paracentesis >250 PMNs; Tx: albumin + ceftriaxone; PPx: FQ

## SAAG
<1.1: Malignancy, Nephrotic syndrome, Pancreatitis, TB
≥1.1: Cirrhosis, Budd-Chiari, R-sided HF

## Ascites Tx
Furosemide or serial paracenteses (give albumin if >5L)

# ACUTE LIVER INJURY
Abrupt ↑AST/ALT + AMS + APAP overdose
Etiologies: APAP, acute viral hepatitis, Amanita mushrooms, systemic hypotension ("shock liver")
Tx: APAP → N-acetylcysteine; Amanita → Penicillin G

# WILSON'S DISEASE
↑AST/ALT + psych sxs + Parkinsonism + Kayser-Fleischer rings
Dx: ↓serum ceruloplasmin; Tx: Penicillamine, Trientine (AE: zinc deficiency)

# PREGNANCY-RELATED GI
1st trimester + severe vomiting + ↑AST/ALT → Hyperemesis gravidarum (Tx: hydration + antiemetics + IV thiamine)
2nd-3rd trimester + pruritus + ↑↑ALT + ↑↑AP → Intrahepatic cholestasis
3rd trimester + BP >140/90 + proteinuria → Pre-eclampsia (Tx: deliver + IV Mag)
3rd trimester + hemolysis + elevated liver enzymes + low platelets → HELLP
3rd trimester + rapid-onset + abnormal liver labs + coma → Acute fatty liver of pregnancy
DDx HELLP vs AFLP: AFLP causes coma/encephalopathy; HELLP does not

# OTHER GI PATHOLOGY
RUQ pain + ↑AST/ALT + fever + no jaundice + thickened GB wall → Cholecystitis (Tx: cholecystectomy)
RUQ pain + jaundice + 104° fever + low BP → Ascending cholangitis (Tx: emergent ERCP + broad-spectrum abx)
Critically-ill + RUQ pain + fever + no gallstones → Acalculous cholecystitis (Tx: cholecystostomy)
Epigastric pain radiating to back + ↑amylase/lipase 3x → Acute pancreatitis (Tx: NPO + pain + IVF)
RUQ pain + purulent vaginal discharge → Fitz-Hugh-Curtis (Tx: ceftriaxone + azithromycin)
Jaundice + dilated CHD + stone in cystic duct → Mirizzi syndrome
Intermittent vomiting + pneumobilia → Gallstone ileus
Elderly + painless large-volume blood → Diverticulosis
65+ + anemia + positive FOBT → Colon cancer
Pt w/ AFib + sudden-onset severe abdominal pain → Acute mesenteric ischemia
Pt w/ AS + microcytic anemia + bloody stools → Heyde Syndrome
Pt w/ telangiectasias + recurrent epistaxis + AV malformations → HHT
