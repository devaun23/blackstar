# DKA

Pt with 2 days n/v, not eating well. Glucose 300, bicarb 5, Na+ 130 -> DKA
Pathophys: Absolute insulin deficiency -> no inhibition of glucagon -> glucagon stimulates ketone body production
Often T1DM (T2DM have enough insulin to inhibit glucagon)
Dx? Ketone bodies (acetoacetate, beta-hydroxybutyrate)
Other causes of ketosis: Starving, Hyperemesis gravidarum, Alcoholics

Electrolyte abnormalities:
- Pseudohyponatremia: Corrected Na+ = measured Na+ + 1.6 x (glucose - 100)/100. Tx? Will correct as you treat
- Hyperkalemia on labs BUT depleted total body K+

Tx:
1. Fluids: 2 L bolus NS @ 1 L/hr -> then 0.45% NS
2. Insulin: K+ must be >3.3 first. If 3.3-5.3 -> add K+ to fluids. If >5.3 -> no extra K+
   - IV Regular insulin 0.1 u/kg/hr
   - Add dextrose when glucose <200
   - Begin SQ insulin: DKA when AG <=12 & glucose 200-250; HHNS when glucose 200-250
   - Turn off infusion 2 hrs after starting SQ insulin
   - Consequence of lowering glucose too fast? Cerebral edema
   - Give bicarb? NO

DKA vs HHNS:
- DKA: Very low bicarb, ketones, T1DM
- HHNS: Normal/near-normal bicarb, no ketones, T2DM


# Hypoglycemia

Causes:
- Injecting insulin -> insulin high, C-peptide low
- Sulfonylurea use -> insulin high, C-peptide high, (+) sulfonylurea screen
  - Mechanism: Block ATP-dependent K+ pump -> depolarization -> Ca+ -> insulin release
- Insulinoma -> insulin high, C-peptide high, (-) sulfonylurea screen

Tx: 50% dextrose then feed. No IV access? IM glucagon
Sulfonylurea hypoglycemia: Observe 24 hrs + octreotide (long half-life -> recurrent episodes)


# Adrenal Crisis

Hypoglycemia + hyponatremia + hyperkalemia + metabolic acidosis -> adrenal crisis
Presentations:
- Pt on steroids that runs out of medication
- Pt on steroids for COPD/asthma with infection (metabolic stress)
- Surgery w/o stress-dose steroids
- Pt with other autoimmune disease + new Addison's
- Meningitis + hypotensive + hyperkalemia + hyponatremia (Waterhouse-Friedrichsen)

CBC findings? Eosinophilia (corticosteroids cause eosinophil apoptosis)

Dx: Morning cortisol + ACTH
- Both low -> secondary adrenal insufficiency (anterior pituitary)
- Elevated ACTH + low cortisol -> primary adrenal insufficiency
- Skin hyperpigmentation in primary: ACTH from POMC -> also MSH -> melanocyte stimulation

Tx? Dextrose + hydrocortisone (has both glucocorticoid & mineralocorticoid properties, better than dexamethasone)


# Thyroid Storm

Pt with hx Graves + missed meds, now severe tachycardia + mild fever + diarrhea + AMS -> thyroid storm
Other triggers: Iodinated contrast, Amiodarone
MC arrhythmia? AFib. Labs: TSH low, T3/T4 high

Tx:
1. Propranolol (hyperadrenergic sxs + blocks peripheral T4->T3 via 5'-deiodinase inhibition)
2. PTU (prevents thyroid hormone synthesis + blocks peripheral conversion)
3. Wait 1 hr -> Lugol's solution/SSKI (Wolff-Chaikoff effect -> prevents thyroid hormone production)
4. Hydrocortisone (treats adrenal insufficiency + prevents peripheral T4->T3 conversion)


# Myxedema Coma

Pt with autoimmune dz hx + profoundly bradycardic + hypothermia + decreased DTRs + periorbital/non-pitting edema -> Myxedema coma
Labs: TSH elevated, T3/T4 decreased
Tx: Levothyroxine (T4) + hydrocortisone (until primary adrenal insufficiency ruled out)
- If hyponatremic -> NS. If hypoglycemic -> dextrose. If very sick -> liothyronine (T3)


# Sheehan's Syndrome

Postpartum woman with failure to lactate. Delivery c/b hemorrhage. May have hypothyroidism signs -> Sheehan's syndrome
Pathophys: Ischemic stroke of pituitary (massive hypertrophy during pregnancy -> susceptible to infarction)
Dx? TRH stimulation test (failure of prolactin to rise -> pituitary insufficiency)


# Pituitary Apoplexy

Sudden HA or visual field deficits + adrenal insufficiency sxs -> pituitary apoplexy
Population: Pts with pituitary adenomas
Pathophys: Hemorrhagic stroke of pituitary ("SAH of pituitary")
Why vision changes? Compression of optic chiasm
Tx: Replace hormones as necessary
