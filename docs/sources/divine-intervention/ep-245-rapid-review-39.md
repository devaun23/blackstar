# SEPTIC ARTHRITIS
Severe joint pain + erythematous → NBS: Arthrocentesis (WBC >50k → washout)
Bug: Staph aureus (sickle cell → Salmonella); Tx: Vancomycin
+ Petechiae on skin → gonococcal; Tx: ceftriaxone/cefotaxime
Negative gram stain → vanc + ceftriaxone
DDx septic arthritis (pain over joint) vs osteomyelitis (tender over bone)

# CERVICAL CANCER SCREENING
21-29: Pap q3 yrs (don't start before 21 even if sexually active)
30+: preferred Pap + HPV co-testing q5 yrs
Immunocompromised (HIV, DiGeorge, h/o high-risk lesions): Paps annually
In utero DES: Paps annually
Hysterectomy for benign reason: can stop Paps
Hysterectomy for malignant reason: Paps of vaginal cuff
Stop at 65 if no hx abnormal findings; conizations increase risk of cervical insufficiency

# SVC SYNDROME
Smoker + severe HA + face appears bulging + distended neck veins → SVC syndrome
Tx: radiation (not chemo for symptom relief); assoc: Small cell lung cancer

# FAMILIAL HYPERTRIGLYCERIDEMIA
Family hx pancreatitis + epigastric pain + no alcohol/gallstones → Type 4 familial dyslipidemia (AD)
Tx: fibrate (gemfibrozil) — MOA: activates PPAR-alpha → more LPL; AE: myopathy, hepatotoxic, gallstones
DM + HFrEF → do NOT give thiazolidinediones (PPAR-gamma → water retention → worsens HF)

# MEMBRANOUS NEPHROPATHY
Generalized edema + 4+ proteinuria + chest pain/SOB/tachycardia + CT filling defect
Pathophys: ATIII lost in urine → can't inhibit factor X/II → hypercoagulable → renal vein thrombosis or PE
Ab against phospholipase A2 receptor; assoc: solid organ cancers, SLE (if lupus + nephritic → DPGN)

# OCPs AND HYPERTENSION
32 yo sexually active female + HTN → OCPs (MCC HTN in reproductive age females)

# SHUNT LOCALIZATION
SVC to RA: biggest jump → ASD
RA to RV: biggest jump → VSD

# DERM PART 2 HIGHLIGHTS (Ep 246)
Leishmaniasis: painless purplish ulcerating papule from sandfly; Dx: skin biopsy; Tx: amphotericin B
Zoster "explosion" → screen for HIV
Molluscum contagiosum outbreak → screen for HIV
Porphyria cutanea tarda → screen for Hep C
Zoster ophthalmicus (V1, tip of nose) → refer to ophthalmologist
Ramsay Hunt: vesicles in ear + CN7 palsy → Tx: acyclovir
Post-herpetic neuralgia → Tx: gabapentin
Scabies: itchy rash between finger webs; Tx: permethrin (NOT lindane — neurotoxic)
Seborrheic keratosis: "stuck-on" appearance; sudden onset of many → screen for GI malignancy (Leser-Trelat sign)
Actinic keratosis: rough sandpaper texture on sun-exposed skin → precursor to SCC; Tx: 5-FU, imiquimod
Biggest RF skin cancer: sun exposure; UVB worse than UVA
SCC: bottom lip, scalp, ear, neck; Keratoacanthoma: volcano-like with keratin
BCC: pink pearly translucent + telangiectasias; spreads horizontally; Tx: Mohs surgery
Melanoma: ABCDE criteria; nodular = worst prognosis; acral lentiginous in African Americans
Familial melanoma dysplastic syndrome: AD
Breslow depth = prognosis; >1mm thick → sentinel lymph node biopsy
Jarisch-Herxheimer reaction: fever/headache/myalgia after treating spirochetes → supportive, continue abx
DRESS syndrome: edema + rash + elevated AST/ALT + eosinophilia + lymphadenopathy → Type IV HSR; Tx: IVIG
