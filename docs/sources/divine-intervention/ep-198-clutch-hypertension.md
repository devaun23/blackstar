# Antihypertensives in Pregnancy
"Hypertensive Moms Love Nifedipine" — Hydralazine, Alpha-methyldopa, Labetalol, Nifedipine
Do NOT give: ACE-I/ARB (renal dysfunction in fetus)

# HTN + Specific Conditions

HTN + stable angina → Nitrate (venodilator → reduce preload → reduce O2 demand)
HTN + migraines → Beta-blocker
HTN + systolic HF w/ peripheral edema → Loop diuretic (ethacrynic acid = no sulfa)
HTN + osteoporosis → Thiazide (reabsorbs Ca++)
HTN + nephrolithiasis → Thiazide (hypercalcemia but hypocalciuria)
HTN + hyperthyroidism → Propranolol (inhibits 5'-deiodinase, T4→T3)
HTN + BPH → alpha-1 blocker (prazosin, doxazosin); tamsulosin (alpha-1b) if don't want to drop BP
HTN + DM or renal insufficiency → ACE-I/ARB (dilate efferent arteriole → prevent hyperfiltration)
HTN + PTSD w/ nightmares → Prazosin
HTN + vasospastic disease (Raynauds) → Dihydropyridine CCB (amlodipine, nifedipine)
HTN + A-fib → Beta-blocker or non-DHP CCB (class 2/4 antiarrhythmic)
HTN 2/2 Conn's → Aldosterone antagonist
HTN + bipolar w/ nephrogenic DI from lithium → ENaC blockers (amiloride, triamterene)
HTN 2/2 pheo → Alpha-blocker (phenoxybenzamine) THEN beta-blocker (beta first → unopposed alpha → crisis)
HTN 2/2 cocaine → Do NOT give beta blocker! Tx: Benzos

# Drugs That Improve Survival in HF
Beta-blockers: metoprolol, carvedilol, ER bisoprolol
ACE-Is/ARBs
Aldosterone antagonists (spironolactone, eplerenone)
BiDil (isosorbide dinitrate + hydralazine) in African Americans

# ACE-I Side Effects and Contraindications

## Dry Cough
Mechanism: ACE also breaks down bradykinin → ACE-I → bradykinin ↑ → dry cough
Site of action: lungs (endothelial cells of lung capillaries)
NBSM: Switch to ARB

## Contraindications
Hereditary angioedema: C1 esterase inhibitor deficiency (also breaks down bradykinin)
Bilateral renal artery stenosis: already low afferent arteriole pressure → RAS → efferent constriction maintains GFR. ACE-I blunts RAS → efferent dilates → filtration drops
Presentation: Cr doubles + hyperkalemia after starting lisinopril

# Thiazide + Hypokalemia
NBS: Add potassium-sparing diuretic — aldosterone antagonists or ENaC blockers

# Drug-Induced Lupus
Triggered by hydralazine → malar rash
Drugs: INH, Procainamide, Hydralazine, Phenytoin, Etanercept
Ab? Anti-histone
CI to hydralazine? Aortic dissection

# Hypertensive Emergency Drugs
Nitroprusside, Labetalol, Nicardipine/clevidipine

## Nitroprusside Toxicity
Cyanide toxicity → lactic acidosis + AMS
Tx: Hydroxocobalamin (B12 derivative) OR Amyl nitrate + sodium thiosulfate

# HTN Drugs Contraindicated in Specific Conditions

CHF exacerbation → avoid contractility reducers (beta blockers, non-DHP CCB)
WPW → avoid AV-nodal blocking agents (BB, CCB, digoxin) — causes more flux through accessory pathway
2nd degree Mobitz II or 3rd degree heart block → avoid beta-blockers
Gout → avoid thiazides & loop diuretics (interfere with uric acid excretion)

# Clonidine (Alpha-2 Agonist)
Miss a dose → rebound hypertension
Indications: Opioid withdrawal, Tourette's (1st alpha-2 agonist → 2nd atypical antipsychotic → 3rd typical antipsychotic)

# Spironolactone Side Effect
Gynecomastia (blocks androgen receptors)

# Nephrogenic DI
Tx: Thiazide diuretic
Exception: lithium toxicity → ENaC blocker (amiloride)
